CIFAR-10 Image Classification System

Description:An image classification system using Convolutional Neural Networks (CNNs) integrated with a React Native front-end and Flask back-end.

Technologies Used

  • Flask: Handled image requests from the mobile app and sent predictions back to the user.
  • TensorFlow: Built and trained the CNN model for classifying images from the CIFAR-10 dataset.
  • Pillow: Managed image preprocessing tasks like resizing and formatting before classification.
  • React Native: Created the mobile interface for users to upload images and view classification results.

This Project integrates deep learning with mobile applications, providing users with real-time image classification functionality in a smooth and interactive way.
